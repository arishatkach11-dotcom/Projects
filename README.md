Это не просто очередной клон судоку. Это **полноценная браузерная игра**, написанная на чистом JavaScript, с уникальной механикой отслеживания использованных цифр и эффектом "исчезающего поля" при победе.

## ✨ Особенности

*   **🧠 Интеллектуальный счётчик цифр**
    *   Каждая кнопка (1-9) показывает, сколько раз цифра уже использована *(например, `5/9`)*.
    *   Когда цифра установлена 9 раз, её кнопка **автоматически блокируется** и становится неактивной — это помогает избежать ошибок и ускоряет игру.

*   **🌓 Две темы оформления**
    *   **Тёмная тема:** Космическая атмосфера с золотыми фиксированными цифрами и голубыми пользовательскими.
    *   **Светлая тема:** Классический чистый вид с чёрными фиксированными цифрами и синими пользовательскими.
    *   Переключение одним кликом (иконка 🌙/☀️).

*   **🎆 Эффект «Исчезновение»**
    *   При заполнении всех ячеек игровое поле красиво анимируется: сжимается и исчезает, открывая окно победы. Сделано на чистом CSS.

*   **🎮 Полноценный геймплей**
    *   Три уровня сложности: Лёгкий, Средний, Сложный.
    *   Система жизней (3 ❤️) и штраф за ошибки.
    *   Таймер и счётчик ходов.
    *   Подсказки (3 штуки), которые вставляют правильную цифру.
    *   Умная подсветка: выделение строки, столбца, блока и всех одинаковых цифр.

## 🕹️ Управление

| Действие | Способ |
| :--- | :--- |
| **Выбрать ячейку** | Клик мышкой по ячейке |
| **Ввести цифру** | Нажать кнопку с цифрой 1-9 **или** клавишу на клавиатуре |
| **Очистить ячейку** | Нажать кнопку «Очистить» **или** клавишу `Backspace` / `Delete` |
| **Получить подсказку** | Нажать кнопку «Подсказка» **или** клавишу `H` |
| **Сменить тему** | Нажать на иконку 🌙/☀️ в верхней панели |

## 🛠️ Технологии

Проект написан с использованием **чистого JavaScript**, без сторонних библиотек или движков.

*   **HTML5** — семантическая разметка.
*   **CSS3** — кастомные свойства, анимации, адаптивность.
*   **JavaScript (ES6+)** — вся логика игры, включая генератор судоку, рендеринг и управление состоянием.
